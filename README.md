# 🌐 Personal Portfolio Website

This is my personal **developer portfolio website** built to showcase my projects, technical skills, and resume. It highlights who I am, what I’ve built, and how to contact me.

## 🚀 Features

- Responsive design for all screen sizes
- Smooth navigation and scroll animations
- Project showcase with GitHub repos
- Skills and technology stack display
- Downloadable resume

## 🛠️ Built With

- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: GitHub Pages
